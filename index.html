Sway hot-keys

Actions

Super + Enter New terminal

Super + F Make current window fullscreen

Super + Shift + Q Quit program

Super + Shift + E Exit Sway

Super + Shift + C Reload Sway configuration

Super + Shift + Space Move window to scratchpad

Super + Space Show scratchpad

Screen capture

PrintScreen Make full screen screenshot to clipboard

Ctrl + PrintScreenMake frame screenshot to clipboard

Alt + PrintScreenMake full screen screenshot to file<

Alt + Ctrl + PrintScreenMake frame screenshot to file<

Workspaces keys

Super + 0..9 Change current workspace

Super + Shift + 0..9 Move current window to designated workspace

Super + B Horizontal layout

Super + V Vertical layout

Super + S Stacking layout

Super + E Toggle split layout

Super + W Tabbed layout

Super + A Focus on parent container

Super + Space Swap focus between tiling and floating

Super + Shift + Space Toggle floating mode

Super + Tab Next workspace

Super + Shift + Tab Previous workspace

Super + Left/Right/Up/Down Move focus of the window

Super + Shift + Left/Right/Up/Down Move the focused window in the workspace

Resize mode

To resize windows, you must enter resize mode, with the Super + R keybinding. After you enter resize mode you can resize windows with Left/Right/Up/Down keys, and press enter or escape to return to default mode.
